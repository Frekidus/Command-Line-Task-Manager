# Command-Line-Task-Manager
 # 📝 Task Manager CLI (Python)

A simple command-line Task Manager application written in Python. This program allows users to create, view, update, and delete tasks, with persistent storage using a text file.

## 🚀 Features

* Add new tasks
* View all tasks
* Mark tasks as complete
* Delete tasks
* Persistent storage using a local file (`tasks.txt`)
